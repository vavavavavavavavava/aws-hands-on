# 事前準備: Gitインストール手順（Windows）

- 戻る: [ハンズオン本編](./handson-board.md)
- 全体導線: [README](../README.md)

## 0. この手順でやること

- Windows に Git をインストール
- コマンドプロンプトで Git 動作確認
- ハンズオンで使うフォルダでコマンドを実行できる状態にする

## 1. Git のインストール

1. ブラウザで `Git for Windows` と検索し、公式サイトを開く
2. `Download` を押してインストーラーを保存する
3. ダウンロードしたインストーラーをダブルクリックする
4. セットアップ画面は基本的に既定値のまま `Next` を進める
5. `Install` を押す
6. 完了後 `Finish` を押す

## 2. Git が使えるか確認

1. `Win` キーを押し、`cmd` と入力してコマンドプロンプトを開く
2. 次を実行する

```bash
git --version
```

1. バージョンが表示されればインストール完了

## 3. ハンズオン用フォルダで実行する方法

1. エクスプローラーでプロジェクトフォルダ（`README.md` がある場所）を開く
2. フォルダ内の空白部分を右クリックする
3. `ターミナルで開く`（または `コマンドプロンプトをここで開く`）を選ぶ
4. 次を実行して、`README.md` が見えることを確認する

```bash
dir
```
