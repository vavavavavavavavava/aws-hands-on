# 事前準備: AWSアカウント作成手順

- 戻る: [ハンズオン本編](./handson-board.md)
- 全体導線: [README](../README.md)

## 0. この手順でやること

- AWS アカウントを作成
- マネジメントコンソールにログイン
- ハンズオンで使うリージョンを確認

## 1. 事前に用意するもの

1. メールアドレス（受信できるもの）
2. 電話番号（SMS または音声認証に使う）
3. クレジットカードまたはデビットカード
4. 本人確認情報（氏名、住所）

## 2. AWS アカウント作成

1. ブラウザで `AWS アカウント 作成` と検索し、AWS公式のサインアップページを開く
2. `AWS アカウントを作成` を押す
3. メールアドレス、アカウント名、パスワードを入力する
4. メール認証コードが届いたら入力する
5. 連絡先情報（個人/法人、氏名、住所、電話番号）を入力する
6. 支払い情報を入力する
7. 電話認証（SMS または音声）を完了する
8. サポートプランは学習用途なら `Basic Support` を選択する
9. 作成完了後、`マネジメントコンソールにサインイン` を押す

## 3. 初回ログイン確認

1. サインイン後、画面右上のアカウント名が表示されることを確認する
2. 画面右上のリージョン選択で、ハンズオン用リージョン（例: `Asia Pacific (Tokyo)`）を選ぶ
3. ブラウザを閉じず、次は [ハンズオン本編](./handson-board.md) の「1. 事前準備」に戻る

## 4. つまずいたとき

1. 認証メールが届かない場合: 迷惑メールフォルダを確認する
2. 電話認証が失敗する場合: 電話番号の国番号を確認して再実行する
3. サインインできない場合: パスワードリセットを実施する
